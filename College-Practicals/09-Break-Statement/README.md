# Break Statement

This Java program demonstrates the use of the `break` statement by searching for a number in an array.

## ⚙️ How It Works

1. Stores a list of numbers in an array.
2. Accepts a number from the user.
3. Searches the array using a loop.
4. Stops the loop immediately when the number is found using the `break` statement.
5. Displays whether the number exists in the array.

- Scanner Class
- Arrays

- break Statement
