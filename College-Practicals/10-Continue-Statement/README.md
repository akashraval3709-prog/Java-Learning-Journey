# Continue Statement

This Java program demonstrates the use of the `continue` statement by printing only odd numbers within a given range.

## ⚙️ How It Works

1. Accepts a limit from the user.
2. Uses a `for` loop to iterate from 1 to the given limit.
3. Skips even numbers using the `continue` statement.
4. Displays only odd numbers.

## 📚 Concepts Used

- Scanner Class
- Variables
- User Input
- For Loop
- continue Statement
- Conditional Statements
- Modulus Operator
- Console Output
