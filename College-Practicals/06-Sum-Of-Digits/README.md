# ➕ Sum of Digits

This Java program calculates the sum of all digits of a number entered by the user.

## ⚙️ How It Works

1. 📝 Accepts an integer from the user.
2. 🔢 Extracts each digit using the modulus (`%`) operator.
3. ➕ Adds each digit to the sum.
4. 📢 Displays the final sum of all digits.

## 📚 Concepts Used

* 📥 Scanner Class
* ⌨️ User Input
* 🔁 While Loop
* ➗ Modulus Operator (`%`)
* ➖ Division Operator (`/`)
* 🔢 Variables
* 🖥️ Console Output
