# Factorial of a Number

This Java program calculates the factorial of a given number using a loop.

A factorial is the product of all positive integers from **1** to the given number.

**Example:**

* 5! = 5 × 4 × 3 × 2 × 1 = 120

## Concepts Used

* Scanner Class
* Variables
* While Loop
* Arithmetic Operations
* User Input
* Output Statements
