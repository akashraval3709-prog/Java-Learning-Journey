# 🔢 Array Sorting

This Java program sorts the elements of an array in **ascending order** using the **Bubble Sort** algorithm.

## ⚙️ How It Works

1. Accepts the size of the array from the user.
2. Reads the array elements.
3. Displays the original array.
4. Sorts the array using the Bubble Sort algorithm.
5. Displays the sorted array in ascending order.
