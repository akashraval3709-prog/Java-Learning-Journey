# 📝 Parameterized Constructor

This Java program demonstrates the use of a **parameterized constructor** to initialize object data using values provided by the user.

## ⚙️ How It Works

1. Accepts employee details from the user.
2. Passes the values to the parameterized constructor.
3. Initializes the object using the constructor.
4. Displays the employee information.
