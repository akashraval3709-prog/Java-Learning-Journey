# 💻 Command-Line Arguments

This folder contains Java programs that demonstrate the use of **command-line arguments** in Java through practical examples.

## ⚙️ Programs

### 1. Greeter.java

Displays a personalized greeting when a name is provided as a command-line argument. Otherwise, it displays a default greeting.

### 2. DynamicCalculator.java

Performs arithmetic operations using command-line arguments. The program accepts multiple numbers and an operator (`+`, `-`, `*`, `/`) to calculate the result.

## ⚙️ How It Works

1. Reads input using the `args[]` array.
2. Processes command-line arguments.
3. Performs the requested operation based on the provided arguments.
4. Displays the calculated result or an appropriate error message.

## 📚 Concepts Used

- Command-Line Arguments
- `args[]` Array
- Arrays
- Methods
- Switch Statement
- Loops
- Arithmetic Operators
- Conditional Statements
- Type Conversion (`Integer.parseInt()`, `Double.parseDouble()`)
- Console Output
