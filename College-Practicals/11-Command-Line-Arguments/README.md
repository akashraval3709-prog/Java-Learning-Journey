# 💻 Command-Line Arguments

This Java program demonstrates how to use **command-line arguments** in Java. If a user's name is provided as a command-line argument, the program displays a personalized greeting. Otherwise, it displays a default greeting.

## ⚙️ How It Works

1. Reads the command-line arguments using the `args[]` array.
2. Checks whether a name has been provided.
3. Displays a personalized greeting when an argument is available.
4. Displays a default greeting when no arguments are passed.

## 📚 Concepts Used

- Command-Line Arguments
- `args[]` Array
- Arrays
- `if-else` Statement
- String Handling
- Console Output
