# 🧬 Single Inheritance

This Java program demonstrates the concept of **Single Inheritance**. A `SavingsAccount` class inherits the properties and methods of the `BankAccount` class and adds its own functionality to calculate interest.

## ⚙️ How It Works

1. Accepts bank account details from the user.
2. Creates a `SavingsAccount` object.
3. Inherits account details from the `BankAccount` class.
4. Calculates and displays the interest earned.
5. Displays the complete account information.
