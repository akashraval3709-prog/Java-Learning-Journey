# ⚡ Program 16 - Nesting of Methods

This Java console application demonstrates the concept of **Nesting of Methods**.

The program calculates an electricity bill by calling one method from another. It computes the unit charge, calculates GST, and then generates the final bill using nested method calls.

## 📚 Concepts Covered

- Methods
- Nesting of Methods
- Static Methods
- Method Calling
- User Input using Scanner

## 🛠️ Program Flow

1. Enter the number of electricity units.
2. Calculate the unit charge.
3. Calculate GST.
4. Calculate the total bill.
5. Display the final bill.

## 📌 Example Output

```text
Enter Units Consumed : 120

----- Electricity Bill -----
Units Consumed : 120
Total Bill     : ₹1132.8
```

## 🎯 Learning Objective

- Understand how one method can call another method.
- Learn method reusability.
- Practice modular programming in Java.
