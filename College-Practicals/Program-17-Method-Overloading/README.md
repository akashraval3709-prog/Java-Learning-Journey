# Program-17: Method Overloading in Java

This program demonstrates the concept of **Method Overloading** in Java by calculating the area of different shapes using methods with the same name but different parameter lists.

## 📚 Objective

- Understand Method Overloading
- Learn compile-time polymorphism
- Calculate the area of different geometric shapes

## 🛠️ Concepts Covered

- Method Overloading
- Static Methods
- User Input using Scanner
- Arithmetic Operations
- Compile-Time Polymorphism

## ▶️ Methods

- `area(int side)` → Calculates the area of a square.
- `area(int length, int width)` → Calculates the area of a rectangle.
- `area(double radius)` → Calculates the area of a circle.

## 📥 Sample Input

```text
Enter Side of Square : 5
Enter Length : 10
Enter Width : 8
Enter Radius : 7
```

## 📤 Sample Output

```text
Area of Square : 25
Area of Rectangle : 80
Area of Circle : 153.86
```

## 🎯 Learning Outcome

After completing this program, you will understand how Java selects the appropriate overloaded method based on the number and type of arguments passed.
