# Prime Number Check

This Java program checks whether a given number is prime or not.

## ⚙️ How It Works

1. Accepts an integer from the user.
2. Checks whether the number has any divisors.
3. Displays whether the number is Prime or Not Prime.

## 📚 Concepts Used

- Scanner Class
- Variables
- User Input
- For Loop
- Conditional Statements
- Comparison Operators
- Console Output
