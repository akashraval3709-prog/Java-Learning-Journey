# ➗ Sum of Harmonic Series

This Java program calculates the sum of the Harmonic Series up to a given number **n**.

## ⚙️ How It Works

1. Accepts a positive integer (`n`) from the user.
2. Uses a `for` loop to calculate the series:

   * `1 + 1/2 + 1/3 + ... + 1/n`
3. Displays the final sum of the Harmonic Series.

## 📚 Concepts Used

* Scanner Class
* Variables
* User Input
* For Loop
* Arithmetic Operations
* Floating-Point Calculation
* Console Output
