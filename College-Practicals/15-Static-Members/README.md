# 🏦 Static Members

This Java program demonstrates the use of **static members** by creating multiple bank accounts. It uses static variables and static methods to store information shared among all objects.

## ⚙️ How It Works

1. Accepts the number of bank accounts.
2. Reads the account holder's name and initial balance.
3. Creates a `BankAccount` object for each account.
4. Displays the account details.
5. Displays the total number of accounts using a static method.
