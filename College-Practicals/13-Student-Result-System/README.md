# 🎓 Student Result System

This Java program creates a `Student` class to generate a student's result by calculating the total marks, percentage, and grade based on the marks entered by the user.

## ⚙️ How It Works

1. Accepts the number of students.
2. Reads each student's details and marks.
3. Calculates the total marks.
4. Calculates the percentage.
5. Determines the grade based on the percentage.
6. Displays the complete result of each student.
