# 💼 Employee Salary Slip

This Java program creates an `Employee` class to generate a salary slip by calculating allowances, deductions, gross salary, and net salary based on the employee's basic salary.

## ⚙️ How It Works

1. Accepts the number of employees.
2. Reads employee details and basic salary.
3. Calculates allowances (Dearness, House Rent, and Medical).
4. Calculates deductions (Provident Fund and Tax).
5. Computes the gross salary and net salary.
6. Displays a formatted salary slip for each employee.
