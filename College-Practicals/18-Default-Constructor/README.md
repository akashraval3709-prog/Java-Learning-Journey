# 🏗️ Default Constructor

This Java program demonstrates the use of a **default constructor**. When an object is created, the constructor automatically initializes the object with default values.

## ⚙️ How It Works

1. Creates a Student object.
2. Invokes the default constructor automatically.
3. Initializes default values.
4. Displays the student information.
