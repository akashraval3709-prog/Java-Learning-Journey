# 🔢 Largest of Three Numbers

This Java program compares three numbers entered by the user and displays the largest number using the conditional (ternary) operator.

## ⚙️ How It Works

1. 📝 Accepts three integer values from the user.
2. 🔍 Compares the numbers using the ternary operator.
3. 🏆 Displays the largest number.

## 📚 Concepts Used

* 📥 Scanner Class
* 🔢 Variables
* ⌨️ User Input
* ❓ Ternary Operator (`?:`)
* ⚖️ Comparison Operators
* 🖥️ Console Output
