# Largest of Two Numbers

This Java program compares two numbers entered by the user and displays the larger number using an `if-else` statement.

## How It Works

1. Accepts two integer values from the user.
2. Compares both numbers using the `if-else` statement.
3. Displays the larger number.

## Concepts Used

* Scanner Class
* Variables
* User Input
* `if-else` Statement
* Comparison Operators
* Console Output
